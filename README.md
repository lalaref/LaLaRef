## 專案簡介

「球證急單王」是一個專為私人籃球比賽、友誼賽設計的球證預訂網站。專接即日或近期訂單，提供專業持牌核心團隊服務，確保技術質素。

## 主要特色

### 🎯 核心功能
- **即日急單服務** - 專門處理即日或近期的緊急訂單
- **全港體育館覆蓋** - 包含九龍、港島、新界、葵青、離島所有主要體育館
- **專業持牌團隊** - 核心團隊均持專業裁判牌照
- **WhatsApp 即時確認** - 填表後直接透過 WhatsApp 發送預訂資訊

### 📋 預訂表格功能
1. **比賽日期時間選擇**
   - 日期選擇（最早為當天）
   - 時間選擇
   - 時長選擇（1-4小時）

2. **場地選擇**
   - 下拉式選單包含全港所有主要體育館
   - 按地區分類（九龍、港島、新界、葵青、離島）
   - 可填寫場地詳情（場號等）

3. **球證需求**
   - 選擇需要的球證數目（1-4位）
   - 填寫每小時報價預算
   - 自動計算預計總費用

4. **聯絡資料**
   - 姓名
   - WhatsApp 電話號碼
   - 備註欄位

### 💬 WhatsApp 整合
- **浮動按鈕** - 頁面右下角固定 WhatsApp 查詢按鈕（含無障礙標籤）
- **自動訊息生成** - 提交表格後自動生成格式化的預訂訊息
- **一鍵發送** - 點擊提交後直接開啟 WhatsApp 並填入訊息
- **安全連結** - 使用 noopener/noreferrer 保護用戶安全

## 使用方式

### 部署
1. 將所有檔案上傳到網頁伺服器
2. 確保 `index.html` 為首頁
3. 無需資料庫或後端配置

### 自訂設定
在 `index.html` 中修改 WhatsApp 號碼：
```javascript
const whatsappURL = `https://wa.me/85284828484?text=${encodedMessage}`;
```

將 `85284828484` 改為您的 WhatsApp 號碼（包含國碼）。

### 修改體育館列表
在 `index.html` 的 `<select id="venue">` 區塊中新增或修改體育館選項。

### 常見問題

**Q: 焦點輪廓不可見？**  
A: 檢查瀏覽器縮放為 100%，驗證 CSS 未被覆蓋

**Q: 表單驗證不工作？**  
A: 清除瀏覽器快取並重新載入，檢查 JavaScript 控制台

**Q: WhatsApp 未開啟？**  
A: 檢查彈出視窗攔截器設定，驗證 WhatsApp 已安裝
